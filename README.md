# Claude Skills 中文版

> 翻译 Claude 的 Skills 仓库

## 项目简介

本项目是 [Anthropic's Claude Skills](https://github.com/anthropics/skills) 的中文本地化版本。我们定期从原始仓库同步最新内容，并将其翻译成中文，旨在为中文用户提供更便捷的 Claude Skills 学习与使用体验。

## 翻译状态

| ------------ | ----------------------- |
| 最近同步时间 | 2025-01-21              |
| 翻译状态     | AI 翻译完成，待人工校验 |

## 项目结构

本项目保持与原始仓库相同的目录结构，但将内容翻译为中文。主要内容位于 `cn` 目录下，而原始英文内容保存在 `official` 目录中作为参考。

## 贡献指南

我们欢迎社区成员参与到翻译校对工作中。如果您发现任何翻译错误或有改进建议，请随时提交 Issue 或 Pull Request。
