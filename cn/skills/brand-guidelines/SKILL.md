# Anthropic 品牌样式

## 概述

使用此技能可以访问 Anthropic 的官方品牌标识和样式资源。

**关键词**：品牌建设、企业标识、视觉标识、后期处理、样式设计、品牌颜色、排版、Anthropic 品牌、视觉格式、视觉设计

## 品牌指南

### 颜色

**主要颜色：**

- 深色：`#141413` - 主要文本和深色背景
- 浅色：`#faf9f5` - 浅色背景和深色背景上的文本
- 中灰色：`#b0aea5` - 次要元素
- 浅灰色：`#e8e6dc` - 微妙背景

**强调颜色：**

- 橙色：`#d97757` - 主要强调色
- 蓝色：`#6a9bcc` - 次要强调色
- 绿色：`#788c5d` - 第三强调色

### 排版

- **标题**：Poppins（备用 Arial）
- **正文**：Lora（备用 Georgia）
- **注意**：为获得最佳效果，字体应预先安装在您的环境中

## 功能

### 智能字体应用

- 将 Poppins 字体应用于标题（24pt 及更大）
- 将 Lora 字体应用于正文
- 如果自定义字体不可用，自动退回到 Arial/Georgia
- 保持跨所有系统的可读性

### 文本样式

- 标题（24pt+）：Poppins 字体
- 正文：Lora 字体
- 基于背景的智能颜色选择
- 保留文本层次和格式

### 形状和强调色

- 非文本形状使用强调色
- 在橙色、蓝色和绿色强调色之间循环
- 在保持品牌一致的同时维持视觉兴趣

## 技术细节

### 字体管理

- 在可用时使用系统安装的 Poppins 和 Lora 字体
- 自动退回到 Arial（标题）和 Georgia（正文）
- 无需字体安装 - 可与现有系统字体配合使用
- 为获得最佳效果，请在您的环境中预先安装 Poppins 和 Lora 字体

### 颜色应用

- 使用 RGB 颜色值进行精确的品牌匹配
- 通过 python-pptx 的 RGBColor 类应用
- 在不同系统中保持颜色保真度
